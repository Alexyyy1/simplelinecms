SimpleLineCMS

Легкая CMS без базы данных для простых одностраничных сайтов

Что это?

SimpleLineCMS — это минималистичная CMS, которая хранит контент в JSON-файлах и позволяет создавать красивые одностраничные сайты с секциями. Простая в использовании, не требует баз данных.

Основные возможности

  • Редактирование страниц через удобную админку
  • Хранение данных в JSON — никаких баз данных
  • Легкое добавление секций: текст, заголовки, изображения и кнопки
  • Адаптивный и современный дизайн
  • Бесплатно и открытое исходное ПО

Как начать?

  1. Скачайте архив
  2. Распакуйте файлы на ваш веб-сервер с поддержкой PHP
  3. Откройте /admin/edit.php?page=home для редактирования первой страницы
  4. Создавайте и публикуйте ваши сайты легко и быстро!
